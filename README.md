# Девичник — Ляйсян 💕

Одностраничный анимированный сайт-приглашение на девичник.

🔗 **Сайт:** [lyaysyan-devichnik.online](https://lyaysyan-devichnik.online)

## Структура проекта

```
devichnik/
├── index.html        — главная страница
├── css/
│   └── style.css     — все стили
├── js/
│   └── script.js     — вся логика (курсор, конфетти, таймер, слайдер, фейерверк)
└── images/
    ├── photo1.jpg … photo10.jpg   — фотографии в галерее
    └── og-image.jpg               — превью при отправке ссылки (Open Graph)
```

## Технологии

- HTML5 / CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (без зависимостей)
- Google Fonts (Playfair Display, Cormorant Garamond, Montserrat, Great Vibes)
- Yandex Maps Widget (embed)
- Open Graph / Twitter Card meta-теги для превью ссылок

## Возможности

- Адаптивная вёрстка (desktop / mobile)
- Кастомный курсор с поддержкой touch-устройств
- Canvas-анимация конфетти
- Обратный отсчёт до даты события
- Программа мероприятия с таймлайном
- Цветовая палитра дресс-кода
- Слайдер фотографий со свайпом, точками-индикаторами и подписями
- Лайтбокс для просмотра фото
- Встроенная карта (Yandex Maps)
- Canvas-анимация фейерверка
- Красивое превью при отправке ссылки в мессенджерах

## Запуск

Просто открой `index.html` в браузере — никакого сервера не требуется.

## Деплой

Сайт развёрнут на GitHub Pages с привязанным доменом (файл `CNAME`).
